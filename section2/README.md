####4. 🐰 Git을 배워야 하는 이유  

1) Git은 프로젝트의 시간과 차원을 자유롭게 넘나들수 있도록 해줍니다.
시간 - 프로젝트의 버전을 과거로 되돌리거나 특정 내역을 취소할 수 있습니다.
차원 - 프로젝트의 여러 모드를 쉽게 전환하고 관리할 수 있습니다.
2) Git은 여러 사람들이 프로젝트에서 협업할 수 있도록 도와줍니다.
